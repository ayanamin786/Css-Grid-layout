# 📊 Responsive Dashboard Layout (CSS Grid)

A modern and fully responsive dashboard UI built using **HTML5** and **CSS Grid**.  
This project demonstrates how to create a clean, structured layout with a sidebar, header, content sections, and footer — all responsive across devices.

---

## 🚀 Features

- ✅ Fully Responsive Design (Mobile, Tablet, Desktop)
- ✅ CSS Grid Layout System
- ✅ Clean & Modern UI
- ✅ Sidebar + Header + Footer Structure
- ✅ Flexible Content Sections (Analytics, Activity, Statistics)
- ✅ Easy to Customize

---

## 📱 Responsive Breakpoints

| Device        | Layout Behavior |
|--------------|---------------|
| Desktop      | 3-column layout |
| Tablet       | 2-column layout |
| Mobile       | Single column stack |
